# 🎧 AudioRecorderApp (系统音频捕捉引擎)

这是一个基于 Python 的现代轻量级系统音频录制工具。它剥离了复杂的音频工程设置，提供了一个干净直观的界面，适合快速记录电脑发出的任何声音。

## ✨ 核心特性

* **现代纯白 UI**：基于 PyQt6 构建，界面清爽，防误触设计，彻底禁用无效的最大化操作。
* **专业级底层**：抛弃易冲突的旧框架，采用 `pyaudiowpatch` 直接对接 Windows WASAPI 接口。**不卡顿、不死锁**，即使在没有播放声音时点击录制，也不会导致程序崩溃。
* **实时音频反馈**：内置动态 VU 音量条，随时确认是否成功拾取到声音。
* **全局快捷键**：支持后台操作，按下 `F9` 开始/暂停录制，按下 `F10` 停止并保存。
* **智能防呆机制**：自动检测系统环境，如果未安装 FFmpeg，会自动锁定为无损 WAV 格式以保护录制流程。
